Chess Ai written to run on CP400 Calculator with hollyhock3.  
To use for yourself just download Chess.hh3  
I reccomend using depth 4 or 5 for a balanced difficulty however the higher the depth is set to the longer it will take to calculate moves.  
I have not yet benchmarked it for an ELO but i would guess 1500 to 2000 on depth 5.  
This is only a rough protype and will be improved in the future.
